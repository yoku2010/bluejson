Welcome to blue JSON.

Currently we have three utility files:

1. blue.json.array.utility.js
2. blue.json.jquery.utility.js
3. blue.json.string.utility.js

These utility files will give you functions to make your work easy.

Array Utility

This Library extand Array object and add new properties like sum, newSlice and empty.

jQuery Utility

This Library extand jQuery Object and add refresh function which gives you updated selector.

String Utility

This Library extand String object and add new properties like slugify, toProperCase and toTitleCase.
